# Divorce Rate Tracker - India

A comprehensive web application for tracking and analyzing divorce rates in India by comparing marriage and divorce statistics across different states, time periods, and demographics.

## Features

- 📊 **Statistical Dashboard** - Visualize marriage and divorce rates over time
- 🗺️ **State-wise Analysis** - Compare divorce rates across Indian states
- 📈 **Trend Analysis** - Track trends in marriage and divorce statistics
- 📋 **Data Reports** - Generate detailed reports on divorce statistics
- 🔍 **Search & Filter** - Find specific data by state, year, or demographic
- 📱 **Responsive Design** - Mobile-friendly interface

## Technologies

- **Frontend**: React, Chart.js, Tailwind CSS
- **Backend**: Node.js, Express, MongoDB
- **Data Visualization**: D3.js, Chart.js
- **Development**: Vite, npm

## Getting Started

### Prerequisites
- Node.js (v14+)
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/isaacmatthewalex-oss/App.git
cd App

# Install dependencies
npm install

# Start development server
npm run dev
```

### Building for Production

```bash
npm run build
npm start
```

## Data Source

Data is sourced from:
- Ministry of Statistics and Programme Implementation (MoSPI)
- National Portal of India
- Indian Bureau of Civil Registration
